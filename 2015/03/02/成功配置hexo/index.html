成功配置hexo

折腾了一晚上。终于把hexo给弄出来了。
记录一下使用的方法：
1.用git shell 打开所在hexo目录
2.修改博客内容
3.git shell中输入命令行 hexo generate 生成静态文件
4.输入hexo deploy 布置到github page上
